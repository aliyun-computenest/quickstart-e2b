## 🌟 服务简介

> **E2B 社区版版**——开发者友好的开源沙箱平台，专为 AI 应用、智能体（Agent）和自动化任务打造。支持在完全隔离的 Linux 环境中安全运行任意 Python/Shell 代码，内置文件系统访问、进程管理、网络控制与实时日志输出，助你轻松构建可编程、可调试、可扩展的 AI 执行层！

E2B 社区版版是一个免费、开源的本地化代码执行引擎，适用于 LLM 智能体开发、自动化脚本测试、教育实验等场景。通过简洁的 Web 控制台或 RESTful API，用户可快速创建沙箱实例、上传文件、执行命令，并实时查看输出结果，所有操作均在容器级隔离环境中完成，保障主机系统安全。

## 🚀 部署流程

> ⚡ **5 分钟极速上线，开箱即用！**  
> 借助阿里云计算巢，一键部署 E2B 单机版，省去 Docker 安装、依赖配置、端口映射等繁琐步骤，立即获得一个功能完整的代码沙箱服务！

### 1. 一键创建实例  
访问 [计算巢 E2B 单机版部署页](https://computenest.console.aliyun.com/service/instance/create/cn-hangzhou?type=user&ServiceId=service-318e76fe0ae7464f8d5c)，按页面提示填写基础参数：  
  ![部署参数配置](1.jpg)

### 2. 确认资源配置  
系统将自动生成**费用预估明细**。建议选择 ≥2核4GB 配置以确保多沙箱并发流畅运行，确认后点击 **下一步：确认订单**。

### 3. 启动部署  
在订单确认页核对信息无误，点击 **立即创建**。部署过程约需 2–4 分钟。

### 4. 获取访问地址  
部署完成后，在控制台查看 **E2B 访问地址**：  
  ![服务访问地址](2.jpg)

### 5. 开始使用  
首次使用需远程连接ECS，执行以下命令创建沙箱：
```shell
sudo su
cd /root/infra
make -C packages/shared/scripts local-build-base-template
```

沙箱创建完成后执行命令验证沙箱是否正常工作：
```shell
npm i -g @e2b/cli
e2b template list
```
  ![E2B 控制台界面](3.jpg)
